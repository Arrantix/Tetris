Just a simple little Tetris Clone with an attached AI written in JS without any Deep Q-Learning Algorithms or anything like that.
                                        
  -Rules, and Behaviour solely based on a pointsystem 

  -Hardcoded rewards, punishments and different behaviour for special situations situations.
        
  -Some Situations are: 
  
	Clear only Tetris if possible 
	Avoid holes 
	Prioritize clearing holes before going for Tetris
	Max line-height before clearing single lines etc..
